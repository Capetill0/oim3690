Basic HTML Tags

Common tags you will use when writing HTML:

  • <!DOCTYPE html>: Declares HTML5 document type.
  • <html>: Root element that wraps the entire page.
  • <head>: Metadata container (title, meta, links, scripts).
  • <title>: Page title shown in browser tabs.
  • <meta>: Metadata (charset, viewport, description).
  • <body>: Visible page content (text, images).
  • <h1>–<h6>: Headings from largest to smallest.
  • <p>: Paragraph of text.
  • <a>: Link; use href to point to a URL.
  • <img>: Image; use src and alt.
  • <ul>, <ol>, <li>: Lists and list items.
  • <div> / <span>: Generic block/inline containers.
  • <form>, <input>, <label>, <button>: Form controls.
  • <table>, <tr>, <td>: Tables and cells.
  • <strong> / <em>: Important / emphasized text.
  • <pre> / <code>: Preformatted text and code.
  • <blockquote>: Quoted block of text.

Minimal Example

Below is a simple HTML page example (shown as code):

<!DOCTYPE html>
  <html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width,initial-scale=1">
    <title>My First Page</title>
  </head>
  <body>
    <header><h1>Welcome</h1></header>
    <p>This is a paragraph with a <a href="https://developer.mozilla.org/">link</a>.</p>
    <img src="images/photo.jpg" alt="Description">
    <ul><li>Item one</li><li>Item two</li></ul>
    <form>
      <label for="name">Name:</label>
      <input id="name" type="text" name="name">
      <button type="submit">Send</button>
    </form>
    <footer><p>© 2026</p></footer>
  </body>
  </html>
